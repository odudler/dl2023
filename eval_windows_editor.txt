All q-agents (DDQAgent and CQLAgent, with CNN and FCNN) trained against MinimaxAgent opponent (minimax_depth = 2 and minimax_epsilon = 0.3).
The following parameters were used (see notebook.ipynb):

num_episodes = {'TRAIN': 2000, 'EVAL': 100}
with_options = False
# Agent
agent_type = ...               		# DDQAgent, CQLAgent
agent_network_type = ...            	# CNN, FCNN
# Opponent
opponent_type = 'MinimaxAgent'         	# DDQAgent, DeepQAgent, CQLAgent, MinimaxAgent, RandomAgent
minimax_depth = 2
minimax_epsilon = 0.3
# Same for all agents
epsilon_max = 1
epsilon_min = 0.01
epsilon_decay = 0.9997

The result were evaluated for 1000 episodes and 3 different random seeds.

Observations:
- Q-Agents play deterministically. Two possibilities if they play against each other:
1. One agent always wins regardless of whether he starts the game or not.
2. Agents have a 50% probability of winning, depending on whether they start the game or not.
